NEXORA
 

An all-in-one platform designed to empower students with comprehensive learning tools, career resources, and collaborative features to accelerate their academic and professional success.

🚀 Features
Core Tools & Resources
Chatdoc - AI-powered PDF note generation from documents
Glasp - Article and video summarization into concise notes
Fireflies - Automatic transcription of meetings, interviews, and calls
FreeB - Access to paid books and study materials for free
Roadmap - AI-guided career path from courses to job placement
Aptitude Sprint - 5000+ questions in quantitative, verbal, and logical reasoning
Scholary - PDF summarization into flashcards for quick learning
InternArena - Daily verified internships and jobs for beginners
HackArena - Latest hackathons and tech events information
Poe - Access to multiple AI chatbots in one place
Wolframalpha - Step-by-step solutions for math and science problems
Student Connect - Chat space for students to discuss doubts, exams, and placements
Learning Resources
Video Lectures - Curated YouTube playlists from top educators (CodeWithHarry, WsCube Tech, take U forward)
University Courses - Free courses from Harvard, MIT, Stanford, and other top institutions
Trending Courses - Popular free courses in web development, machine learning, cloud computing, and data science
Interview Preparation - 15,000+ interview questions with AI-powered mock interviews
Community & Collaboration
Student Community - Connect with 100K+ active students worldwide
Study Notes - Community-uploaded handwritten notes for video lectures
Discussion Forums - Platform for academic discussions and doubt clearing
🛠️ Tech Stack
Frontend: React 18, TypeScript, Vite
Styling: Tailwind CSS, ShadCN UI components
Backend: Python,Node.js
Database: Supabase (PostgreSQL, Authentication, Real-time, Security)
State Management: React Query (TanStack)
Routing: React Router DOM
Icons: Lucide React
Forms: React Hook Form with Zod validation
Build Tool: Vite
Package Manager: npm/bun

🚀 Getting Started
Prerequisites
Node.js (v18 or higher)
npm or bun
Installation
Clone the repository


git clone https://github.com/Ayushas19/NEXORA
cd nexora
Install dependencies


npm install
# or
bun install
Set up Supabase

Create a new project on Supabase
Copy your project URL and anon key to src/supabase.js
Run the migrations in supabase/migrations/
Start the development server


npm run dev
# or
bun run dev
Open your browser
Navigate to http://localhost:5173

Build for Production

npm run build
# or
bun run build
